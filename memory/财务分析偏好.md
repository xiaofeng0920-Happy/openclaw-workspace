## 📊 分析内容要求

### 必需分析项目
1. **储蓄账户收支分析**
   - 总收入/总支出统计
   - 收入来源分布
   - 支出分类分布
   - Top 10 收款方排名
   - 大额交易记录（>50,000 HKD）

2. **往来账户收支分析**
   - 资金转入/转出统计
   - USD 转出追踪
   - 待查事项标记
   - 建议核查步骤

3. **整体现金流分析**
   - 月度收支趋势
   - 净现金流计算
   - 资金续航预测
   - 财务健康度评分

### 关键指标统计
| 指标 | 要求 |
|------|------|
| 当前总资产 | 必须显示 |
| 历史最高点 | 必须显示 |
| 历史最低点 | 必须显示 |
| 月均支出 | 必须显示 |
| 资金续航 | 必须显示（月数） |
| 财务健康度 | 评分（1-5 分） |
