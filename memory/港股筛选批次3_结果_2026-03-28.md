## 📋 失败原因分析

在筛选过程中，股票主要因以下原因被排除：

| 原因 | 说明 |
|------|------|
| 无财务数据 | AkShare 无法获取部分股票的财务指标 |
| EPS 无效 | EPS 为负或为空（亏损股） |
| PE 过高 | 市盈率 ≥ 30 |
| ROE 不达标 | 任意一年 ROE ≤ 5% |
| ROIC 不达标 | 任意一年 ROIC ≤ 5% |
| 现金流为负 | 任意一年每股经营现金流 ≤ 0 |
| 负债率过高 | 最新资产负债率 ≥ 50% |

---

## 💡 结论与建议

### 结论
本次筛选**未找到**同时满足所有 5 个条件的港股股票。主要原因：

1. **条件过于严格** - 连续 5 年同时满足 ROE>5%、ROIC>5%、现金流为正的公司非常稀少
2. **数据完整性问题** - 部分港股财务数据在 AkShare 中不完整
3. **市场特性** - 港股市场中很多公司财务指标波动较大

### 建议
1. **放宽条件** - 可考虑：
   - ROE/ROIC 改为 3 年连续
   - 现金流改为 3 年为正
   - PE 上限提高到 40

2. **分步筛选** - 先筛选部分条件，再人工复核

3. **使用其他数据源** - Tushare 港股财务数据不完整，建议结合：
   - 富途牛牛 API
   - 东方财富港股数据
   - 港交所披露易
