## ⚠️ 注意事项

### 1. API Key 管理
```bash
# 不要硬编码在代码中！
# 使用环境变量：
export ALPHA_VANTAGE_KEY="your_key_here"
```

### 2. 网络访问
- 部分国外 API 可能需要代理
- 建议优先使用国内数据源（akshare/tushare）

### 3. 依赖管理
```bash
# 定期更新依赖
pip list --outdated
pip install --upgrade <package>
```

### 4. 回测数据
- Qlib 数据较大（A 股约 10GB）
- 首次下载需要时间
- 建议 SSD 存储
