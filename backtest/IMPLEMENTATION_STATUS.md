## 🚀 使用方法

### 1. 单只股票回测

```bash
python3 backtest/backtester.py --code HK.00700 \
  --start 2023-01-01 \
  --end 2026-03-21
```

### 2. 运行演示

```bash
python3 backtest/backtester.py --demo
```

### 3. Python API

```python
from backtest.backtester import BuffettBacktester

backtester = BuffettBacktester()
result = backtester.run_backtest(
    code='HK.00700',
    start_date='2023-01-01',
    end_date='2026-03-21'
)

# 绘制结果
backtester.plot_result(result)
```

## 🎯 回测逻辑

### 买入条件
```python
if 安全边际 >= 30% and 评分 >= 70:
    买入 → 目标仓位 20-40%
```

### 卖出条件
```python
if 价格 > 内在价值 30%:
    卖出 → 清仓
```

### 仓位管理
```python
目标仓位 = 20% + (安全边际 - 30%) / 100
上限 = 40%
```
