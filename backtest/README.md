# 📈 巴菲特策略回测系统

> "历史不会重复，但会押韵。" —— 马克·吐温

---

## 📖 概述

本系统对巴菲特价值投资策略进行历史回测，验证策略有效性。

### 功能

- ✅ 历史数据获取（AKShare）
- ✅ 巴菲特评分计算
- ✅ 交易信号生成
- ✅ 绩效统计分析
- ✅ 可视化图表

## 📊 回测参数

### 默认配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 初始资金 | 1,000,000 | 100 万 |
| 最大仓位 | 40% | 单只股票上限 |
| 交易成本 | 0.1% | 佣金 + 印花税 |
| 买入阈值 | 30% | 安全边际>30% 买入 |
| 卖出阈值 | 30% | 溢价>30% 卖出 |

### 自定义参数

编辑 `backtest/backtester.py`:

```python
class BuffettBacktester:
    INITIAL_CAPITAL = 1000000  # 初始资金
    MAX_POSITION = 0.40        # 最大仓位
    TRANSACTION_COST = 0.001   # 交易成本
    BUY_MARGIN = 30.0          # 买入安全边际
    SELL_PREMIUM = 30.0        # 卖出溢价
```

## 📈 输出指标

### 收益指标

| 指标 | 说明 |
|------|------|
| 总收益率 | 整个回测期的总收益 |
| 年化收益率 | 年化复利收益 |
| 基准收益 | 买入持有策略收益 |
| 超额收益 | 相对基准的超额收益 |

### 风险指标

| 指标 | 说明 |
|------|------|
| 最大回撤 | 最大资产下降幅度 |
| 波动率 | 年化波动率 |
| 夏普比率 | 风险调整后收益 |

### 交易统计

| 指标 | 说明 |
|------|------|
| 总交易次数 | 买入 + 卖出次数 |
| 胜率 | 盈利交易占比 |
| 盈亏比 | 平均盈利/平均亏损 |
| 平均持仓天数 | 平均持有时间 |

## 🎯 回测逻辑

### 买入条件

1. 安全边际 ≥ 30%
2. 巴菲特评分 ≥ 70
3. 当前仓位 < 目标仓位

### 卖出条件

1. 价格 > 内在价值 30%
2. 基本面恶化（ROE 大幅下降）
3. 发现更好的投资机会

### 仓位管理

```python
目标仓位 = 20% + (安全边际 - 30%) / 100
最大仓位 = 40%
```

## ⚠️ 注意事项

### 数据局限

- **财务数据**: 使用当前数据作为历史代理
- **财报延迟**: 实际财报发布有延迟
- **数据质量**: 历史数据可能不完整

### 回测局限

- **幸存者偏差**: 只回测现存股票
- **前视偏差**: 使用了未来数据
- **交易成本**: 简化了实际成本
