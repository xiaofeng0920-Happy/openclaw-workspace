## 📊 升级内容

### 新权重体系
| 因子 | 原权重 | 新权重 | 变化 |
|------|--------|--------|------|
| 动量 | 40% | **35%** | -5% |
| 价值 | 30% | **25%** | -5% |
| 质量 | 30% | **25%** | -5% |
| **情绪** | 0% | **15%** | **+15%** ⭐ |

### 情绪因子组成
情绪因子 (15%) = 换手率评分 (50%) + 资金流向评分 (50%)

**换手率评分：**
- 计算过去 5 日平均换手率
- 标准化到 0-100 分
- 高换手率 → 高情绪热度

**资金流向评分：**
- 主力资金净流入
- 北向资金流向（A 股）
- 涨跌幅代理（美股/港股）

**核心函数：**
- `calculate_sentiment_score(symbol, market)` - 计算单只股票情绪评分

## 📝 维护说明

### 情绪因子调优

如需调整情绪因子权重，修改 `sentiment_factor.py`：

```python
SENTIMENT_WEIGHT = 0.15  # 情绪因子总权重
TURNOVER_WEIGHT = 0.50   # 换手率在情绪因子中的权重
FLOW_WEIGHT = 0.50       # 资金流向在情绪因子中的权重
```

### 综合评级权重

修改 `get_stock_rating_with_sentiment()` 函数中的权重配置。
