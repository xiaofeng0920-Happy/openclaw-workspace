## 方案 1：官网下载（稍后尝试）

### 下载地址
访问富途 OpenD 官方页面：
- https://www.futunn.com/download/opend
- 或打开富途牛牛 APP → 帮助中心 → OpenD 下载

### 安装步骤
1. 下载对应 Linux 版本
2. 上传到服务器 `~/OpenD/` 目录
3. 解压：`tar -xzf OpenD.tar.gz`
4. 启动：`./OpenD --config ~/.futu/futu_config.ini`
5. 扫码登录
