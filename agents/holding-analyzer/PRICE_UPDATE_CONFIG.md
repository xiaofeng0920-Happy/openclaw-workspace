# 股票价格定时更新配置

**更新日期：** 2026-03-23  
**更新频率：** 每 30 分钟

---

## ⏰ 更新 schedule

| 时段 | 频率 | 推送条件 |
|------|------|---------|
| **交易时间** (09:30-16:00 HK) | 每 30 分钟 | 显著变化 (>3%) 才推送 |
| **美股时间** (21:30-04:00 US) | 每 30 分钟 | 显著变化 (>3%) 才推送 |
| **非交易时间** | 每 30 分钟检查 | 仅记录，不推送 |

---

## 📋 Cron 配置

```bash
# 每 30 分钟检查一次股价
*/30 * * * * cd /home/admin/openclaw/workspace/agents/holding-analyzer && python3 run.py --quick --send-on-change >> /tmp/price-update.log 2>&1
```

---

## 🚀 运行参数

| 参数 | 说明 |
|------|------|
| `--quick` | 快速模式，只更新股价，不获取新闻 |
| `--send-on-change` | 仅当有显著变化时发送推送 |
| `--send` | 无条件发送（用于每日汇总） |

---

## 📊 推送策略

### 避免打扰用户

| 条件 | 动作 |
|------|------|
| 无显著变化 (<3%) | 仅记录日志，不推送 |
| 1-2 只股票变化 >3% | 发送简要提醒 |
| 3+ 只股票变化 >3% | 发送详细报告 |
| 任何股票变化 >5% | 立即推送警报 |
| 期权到期 <3 天 | 单独提醒 |

### 静音时段

| 时段 | 策略 |
|------|------|
| 23:00-07:00 | 仅记录，不推送（除非紧急） |
| 周末 | 仅记录，不推送 |

---

## 📁 输出文件

| 文件 | 说明 |
|------|------|
| `reports/price_update_YYYYMMDD_HHMM.md` | 每次更新的详细报告 |
| `logs/price_update.log` | 运行日志 |
| `data/price_cache.json` | 股价缓存（用于对比） |

## 🔧 管理命令

```bash
# 手动运行一次
python3 run.py --quick --send-on-change

# 查看日志
tail -f /tmp/price-update.log

# 查看 cron 任务
crontab -l | grep price

# 暂停更新
crontab -e  # 注释掉对应行

# 恢复更新
crontab -e  # 取消注释
```
