## 🔍 深度分析

### 为什么测试集 IC=0.3050 但回测亏损？

**1. 数据泄露风险**
- 测试集是随机划分（时间序列相邻）
- 可能存在前视偏差
- 回测是严格时间顺序，无泄露

**2. 因子质量问题**

| 因子类别 | 数据来源 | 预测能力 |
|---------|---------|---------|
| 技术因子（MACD/RSI/布林带） | ✅ 真实行情 | ✅ 有效 |
| 估值因子（PE/ROE） | ❌ 默认值 | ❌ 无区分度 |
| 资金流向（北向/主力） | ❌ 随机模拟 | ❌ 无预测能力 |

**3. 模拟因子拖后腿**

```
特征重要性 Top 5:
1. pe_ratio (998)    ← 默认值，所有股票相同！
2. macd_signal (491) ← 真实有效 ✅
3. price_momentum_10d (485) ← 真实有效 ✅
4. rsi_14 (435)      ← 真实有效 ✅
5. bollinger_position (427) ← 真实有效 ✅
```

**问题：** 重要性最高的 PE 因子用的是默认值，模型学到的"规律"是假的！
