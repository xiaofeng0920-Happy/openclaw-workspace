# 股票日报定时任务配置

## 定时规则

**发送时间：** 每个交易日 早上 9:00（Asia/Shanghai）

**跳过：** 周末（周六、周日）和中国/美国法定节假日

## Cron 表达式

```
0 9 * * 1-5
```

（周一至周五 早上 9 点）

## 任务说明

1. 读取 `agents/stock-daily-agent.md` 中的人设 prompt
2. 调用 `finance-data` 技能获取最新市场数据
3. 生成《每日动量报告》
4. 通过飞书发送给用户（ou_52fa8f508e88e1efbcbe50c014ecaa6e）

## 手动触发命令

```bash
# 手动生成一次报告
openclaw run stock-daily-report
```

## 注意事项

- 美股交易日可能与 A 股不同，需注意美国节假日
- 盘前数据在 8:00-9:30 之间最有参考价值
- 如遇重大突发事件，可临时增加特别报告
