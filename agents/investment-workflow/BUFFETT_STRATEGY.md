## 📊 巴菲特选股标准

| 指标 | 权重 | 满分标准 |
|------|------|---------|
| **ROE 连续性** | 25% | 连续 5 年>15% |
| **ROIC 连续性** | 25% | 连续 5 年>12% |
| **股息率** | 15% | >4% |
| **毛利率稳定性** | 15% | >30% 且波动小 |
| **负债率** | 10% | <50% |
| **护城河** | 10% | 品牌/垄断/成本优势 |
| **总计** | 100% | - |

## 🎯 策略逻辑

### 买入条件
1. 巴菲特评分 > 持仓平均分
2. 不在现有持仓中
3. ROE/ROIC 连续性强
4. 高股息（>4%）优先

### 卖出条件
1. 巴菲特评分 < 65 分（及格线）
2. 连续 2 年 ROE<10%
3. 负债率>50%
4. 无护城河

### 持有条件
1. 巴菲特评分 ≥ 75 分
2. 护城河深厚
3. 长期增长逻辑未变

---

## ⚠️ 注意事项

1. **评分基于公开数据** - 实际数据可能滞后
2. **行业差异** - 科技股通常股息低，能源股通常股息高
3. **周期性** - 周期股 ROE/ROIC 波动大
4. **杠杆 ETF** - 不适合长期持有（如 07709）
