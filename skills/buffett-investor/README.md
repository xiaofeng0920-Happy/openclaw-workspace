# buffett-investor - 巴菲特价值投资技能

基于巴菲特价值投资理念的自动化投资技能，支持股票筛选、动态配置、季度调仓、绩效监控全流程。

## 核心功能

### 1. 股票筛选系统
- **筛选范围**：沪深 300 + 中证 1000 成分股（1300 只）
- **筛选条件**：
  - PE_TTM < 30 倍
  - 近 5 年每年市值 > 30 亿
  - 连续 5 年 ROE > 8% 且 ROIC > 8%
  - 资产负债率 < 50%（60 支严选池）
  - 连续 5 年自由现金流为正（60 支严选池）
- **输出**：177 只优质池 / 60 只严选池

### 2. 市场状态识别
基于沪深 300 指数技术指标和市场广度：

| 市场状态 | 判断条件 |
|---------|---------|
| 🐂 牛市 | 收盘价>MA250 + MA250 斜率>0 + RSI(14)>50 + 广度>60% |
| 🐻 熊市 | 满足 2 个及以上：收盘价<MA250、斜率<0、RSI<40、广度<40% |
| 📊 震荡市 | 不满足牛市或熊市条件 |

### 3. 动态配置策略

| 市场状态 | 股票池 | 仓位 | 行业策略 | 个股筛选 |
|---------|--------|------|---------|---------|
| 🐂 牛市 | 177 支全行业 | 90-100% | 超配周期 +20%、成长 +15% | 高 Beta、PE≤35 |
| 🐻 熊市 | 60 支严选 | 50-70% | 超配防御 +25%、高股息 | 负债率<40%、股息>3% |
| 📊 震荡市 | 120 支中等 | 70-85% | 均衡配置、消费/医药 +5% | ROE 稳定、PE≤25 |

### 4. 调仓机制
- **频率**：月度/季度可选
- **时点**：每月初或财报披露后首个交易日
- **流程**：市场状态判断 → 股票池选择 → 行业权重调整 → 个股筛选 → 仓位确定

### 5. 行业均衡配置
- 13 个主要行业，每行业选 ROE/ROIC 前 2 名
- 等权重分配，避免过度集中

### 6. 绩效监控
- 实时跟踪收益率、回撤、夏普比率
- 风险预警：回撤>15% 触发减仓

## 使用示例

```python
from buffett_investor import StockScreener, DynamicAllocator, Rebalancer

# 初始化
screener = StockScreener()
allocator = DynamicAllocator()
rebalancer = Rebalancer()

# 筛选股票
stock_pool = screener.screen(pe_max=30, roe_min=8, roic_min=8, market_cap_min=30)

# 识别市场状态
market_state = allocator.identify_market_state()

# 获取配置建议
allocation = allocator.get_allocation(market_state, stock_pool)

# 执行调仓
rebalance_plan = rebalancer.generate_plan(allocation, current_holdings)
```

## 核心优势

1. **超额收益显著** - 年化 13.39% vs 沪深 300 -2.29%
2. **风险控制优秀** - 回撤 -27% vs 沪深 300 -41%
3. **夏普比率高** - 63.82% vs 沪深 300 -0.15%
4. **可复制性强** - 规则驱动、标准化流程
5. **财务质量高** - 严格筛选 ROE/ROIC/现金流/负债率

## 适用投资者

- **积极型**：177 支股票池 + 月度调仓
- **稳健型**：60 支股票池 + 季度调仓
- **全场景**：动态配置策略

## 文件清单

- `screener.py` - 股票筛选模块
- `market_state.py` - 市场状态识别
- `allocator.py` - 动态配置策略
- `rebalancer.py` - 调仓执行器
- `monitor.py` - 绩效监控仪表板
- `config.py` - 配置参数
- `utils.py` - 工具函数

## 注意事项

1. 历史回测不代表未来收益
2. 月度调仓会产生交易成本
3. 部分小市值股票可能存在流动性风险
4. 建议设置最大回撤>15% 触发减仓
