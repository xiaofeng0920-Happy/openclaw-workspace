| 风控项 | 限制 | 说明 |

## 七、实施建议

### 7.1 动态调整机制

| 频率 | 任务 | 说明 |
|------|------|------|
| **每日** | 监控因子 IC 值 | 发现因子失效 |
| **每周** | 检查行业偏离度 | 控制行业集中度 |
| **每月** | 评估市场情景 | 调整因子权重 |
| **每季** | 重新筛选股票池 | 更新质量因子 |

### 7.2 监控指标

| 指标 | 阈值 | 动作 |
|------|------|------|
| **因子 IC 值** | < 0.03 | 降低该因子权重 |
| **策略换手率** | > 300%/年 | 降低调仓频率 |
| **行业偏离度** | > 15% | 调整行业权重 |
| **最大回撤** | > 15% | 减仓至 50% |

### 7.3 优化方向

1. **引入机器学习** - XGBoost/LightGBM 优化因子权重
2. **增加风险模型** - Barra 风险模型控制风格暴露
3. **结合宏观因子** - GDP、CPI、利率等择时
4. **扩展数据源** - 舆情数据、分析师预期
