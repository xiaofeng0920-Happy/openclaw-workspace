## 🚀 快速开始

### 方式 1：Python 调用

```python
from buffett_investor import StockScreener, MarketStateIdentifier

# 筛选股票
screener = StockScreener()
result = screener.screen(pool_type="60")
print(f"筛选出{result['count']}只股票")

# 识别市场状态
identifier = MarketStateIdentifier()
market = identifier.identify_market_state()
print(f"市场状态：{market['market_state_cn']}")
```

## 📋 配置说明

### 核心参数 (`config.py`)

```python
# 筛选条件
PE_MAX = 30          # PE 上限
ROE_MIN = 8          # ROE 下限%
ROIC_MIN = 8         # ROIC 下限%

# 动态配置
BULL_POSITION = 0.95      # 牛市仓位
BEAR_POSITION = 0.60      # 熊市仓位
OSCILLATE_POSITION = 0.75 # 震荡市仓位

# 风控
MAX_DRAWDOWN_THRESHOLD = 0.15  # 回撤阈值
```
