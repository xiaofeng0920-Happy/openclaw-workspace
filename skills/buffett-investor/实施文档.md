## 策略概述

- ✅ **超额收益显著** - 年化 13.39% vs 沪深 300 -2.29%

- ✅ **可复制性强** - 规则驱动、标准化流程

## 核心组件

### 1. 股票筛选系统

**筛选条件：**
- PE_TTM < 30 倍
- 近 5 年每年市值 > 30 亿
- 连续 5 年 ROE > 8% 且 ROIC > 8%
- （60 支池）资产负债率 < 50%
- （60 支池）连续 5 年自由现金流为正

**股票池：**
- **177 支池**：基础筛选，行业覆盖全面
- **60 支池**：严格筛选，财务质量最优
- **120 支池**：中等筛选，均衡配置

### 2. 市场状态识别

**技术指标：**
- MA250（年线）
- MA250 斜率
- RSI(14)
- 市场广度（涨跌家数比）

**判断规则：**

| 市场状态 | 判断条件 |
|---------|---------|
| 🐂 牛市 | 同时满足：收盘价>MA250、斜率>0、RSI>50、广度>60% |
| 🐻 熊市 | 满足 2 个及以上：收盘价<MA250、斜率<0、RSI<40、广度<40% |
| 📊 震荡市 | 不满足牛市或熊市条件 |

### 3. 动态配置策略

| 市场状态 | 股票池 | 仓位 | 行业策略 | 个股筛选 |
|---------|--------|------|---------|---------|
| 🐂 牛市 | 177 支 | 90-100% | 超配周期 +20%、成长 +15% | 高 Beta、PE≤35 |
| 🐻 熊市 | 60 支 | 50-70% | 超配防御 +25%、高股息 | 负债率<40%、股息>3% |
| 📊 震荡市 | 120 支 | 70-85% | 均衡配置 | ROE 稳定、PE≤25 |

### 4. 调仓机制

**频率：**
- 月度：每月初调仓（收益最高）
- 季度：财报披露后调仓（成本低）

**流程：**
```
市场状态判断 → 股票池选择 → 行业权重调整 → 个股筛选 → 仓位确定 → 生成交易指令
```

### 5. 行业均衡配置

**13 个主要行业：**
- 白酒与葡萄酒（6 只）
- 中药（5 只）
- 工业机械（4 只）
- 西药（3 只）
- 电气部件（3 只）
- 铝（2 只）
- 医疗设备（2 只）
- 有线电视（2 只）
- 海港（2 只）
- 煤炭（2 只）
- 基础化工（2 只）
- 生物科技（2 只）
- 服装（2 只）

**配置规则：** 每行业选 ROE/ROIC 前 2 名，等权重分配

### 6. 绩效监控

**监控指标：**
- 累计收益率
- 年化收益率
- 最大回撤
- 夏普比率
- 波动率

**风险预警：**
- 回撤>15% 触发减仓
- 单只股票亏损>10% 止损
- 单只股票盈利>40% 止盈

## 使用示例

### 示例 1：筛选股票

```python
from buffett_investor import StockScreener

# 初始化筛选器
screener = StockScreener(data_source="akshare")

# 筛选 177 支股票池
result = screener.screen(pool_type="177")

print(f"筛选结果：{result['count']} 只股票")
print(f"平均 PE: {result['statistics']['avg_pe']:.2f}倍")
print(f"平均 ROE: {result['statistics']['avg_roe']:.2f}%")

# 展示 TOP 10
for i, stock in enumerate(result['stocks'][:10], 1):
    print(f"{i}. {stock['name']} - ROE:{stock['roe']:.2f}%")
```

### 示例 2：识别市场状态

```python
from buffett_investor import MarketStateIdentifier

# 初始化识别器
identifier = MarketStateIdentifier()

# 识别当前市场状态
result = identifier.identify_market_state()

print(f"市场状态：{result['market_state_cn']}")
print(f"置信度：{result['confidence']:.2%}")

# 获取配置建议
suggestion = identifier.get_allocation_suggestion(result['market_state'])
print(f"建议仓位：{suggestion['position_min']:.0%}-{suggestion['position_max']:.0%}")
```

### 示例 3：动态配置

```python
from buffett_investor import DynamicAllocator

# 初始化配置器
allocator = DynamicAllocator()

# 获取配置建议
allocation = allocator.get_allocation(
    market_state="oscillate",
    stock_pool_data={},
    risk_preference="balanced"
)

print(f"股票池：{allocation['stock_pool']['name']}")
print(f"推荐仓位：{allocation['recommended_position']:.0%}")
print(f"行业权重：{allocation['industry_weights']}")
```

### 示例 4：完整流程

```python
from buffett_investor import StockScreener, MarketStateIdentifier, DynamicAllocator

# 初始化
screener = StockScreener()
identifier = MarketStateIdentifier()
allocator = DynamicAllocator()

# 1. 筛选股票
stock_pool = screener.screen(pool_type="177")

# 2. 识别市场状态
market_state = identifier.identify_market_state()

# 3. 获取配置建议
allocation = allocator.get_allocation(
    market_state=market_state['market_state'],
    stock_pool_data=stock_pool
)

# 4. 输出配置方案
print(f"市场状态：{market_state['market_state_cn']}")
print(f"股票池：{allocation['stock_pool']['name']}")
print(f"仓位：{allocation['position_range']['min']:.0%}-{allocation['position_range']['max']:.0%}")
print(f"推荐持仓：{allocation['stock_filters']['focus']}")
```

## 风险提示

### ⚠️ 重要风险

1. **历史回测不代表未来**
   - 过去 5 年表现优异不代表未来持续
   - 市场环境变化可能影响策略有效性

2. **交易成本**
   - 月度调仓产生佣金和冲击成本
   - 实际收益可能低于回测 1-2%

3. **流动性风险**
   - 部分小市值股票流动性差
   - 大额交易可能影响价格

4. **财务数据滞后**
   - 基于历史财报数据
   - 存在信息滞后性

5. **最大回撤**
   - 策略最大回撤约 -25%~-29%
   - 需做好心理准备和资金管理

### ✅ 风险控制措施

1. **仓位管理**
   - 单只股票最大仓位 5%
   - 单行业最大仓位 20%

2. **止损止盈**
   - 单只股票止损线 10%
   - 单只股票止盈线 40%

3. **回撤控制**
   - 组合回撤>15% 触发减仓
   - 减仓比例 20%

4. **分散投资**
   - 13 个行业分散
   - 至少 20-30 只股票
