# 股票持仓监控技能

## 描述
监控用户股票持仓价格变动，提供实时股价跟踪、涨跌幅提醒、持仓盈亏分析等功能。

**触发条件：**
- 用户询问持仓股票价格
- 用户要求监控某只股票
- 定时监控任务

**支持功能：**
1. 📊 实时股价查询
2. 📈 涨跌幅监控
3. 💰 持仓盈亏计算
4. 🔔 价格提醒
5. 📉 历史价格趋势

---

## 执行步骤

### 步骤 1：获取持仓信息
- 询问用户持仓股票列表
- 记录每只股票的：代码、名称、持仓数量、成本价

### 步骤 2：查询实时价格
- 调用金融数据 API 查询股价
- 计算涨跌幅
- 计算持仓盈亏

### 步骤 3：生成监控报告
- 持仓概览表
- 盈亏排行
- 异常波动提醒

### 步骤 4：定期更新
- 每日收盘后更新
- 重大波动时提醒

---

## 数据模板

### 持仓记录
| 字段 | 说明 |
|------|------|
| 股票代码 | 如 600519.SH |
| 股票名称 | 如 贵州茅台 |
| 持仓数量 | 股数 |
| 成本价 | 买入均价 |
| 当前价 | 实时价格 |
| 涨跌幅 | 当日/累计 |
| 持仓盈亏 | 金额和百分比 |

---

## 注意事项

1. 数据延迟：A 股/港股数据可能延迟 15 分钟
2. 不是投资建议：仅供参考，不构成投资建议
3. 风险提示：股市有风险，投资需谨慎
