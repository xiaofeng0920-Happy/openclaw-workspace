# 财务分析技能

## 描述
银行结单财务分析技能。支持分析中信银行国际账户结单，生成完整的财务分析报告，包括 Excel 数据表、HTML 图表、PDF 报告等。

**触发条件：**
- 用户提供银行结单（PDF 格式）
- 用户询问财务分析、收支统计、现金流分析等
- 用户要求生成财务报告、Excel 报表、PDF 报告等

**支持功能：**
1. 📊 银行结单数据提取
2. 📈 收支统计分析
3. 💰 现金流预测
4. 📉 财务健康度评估
5. 📋 生成 Excel/PDF/HTML 报告
6. 🔍 大额交易追踪
7. 👥 收款方分析

---

## 执行步骤

### 步骤 1：接收结单文件
- 用户发送银行结单 PDF 文件
- 确认文件数量和账户信息
- 提取关键数据（日期、余额、交易记录）

### 步骤 2：数据整理与统计
- 按账户分类（储蓄账户、往来账户）
- 统计月度收支
- 计算关键指标（总收入、总支出、净现金流等）

### 步骤 3：生成分析报告
调用子代理生成以下文件：
1. **Excel 报表** - 完整数据表（多个工作表）
2. **HTML 图表** - 交互式可视化图表
3. **PDF 报告** - 正式分析报告
4. **Markdown/HTML 摘要** - 简洁版报告

### 步骤 4：发送文件给用户
- 通过用户当前使用的渠道发送所有文件
- 提供核心数据摘要
- 说明每个文件的内容和用途

### 步骤 5：后续支持
- 回答用户关于报告的问题
- 根据需求调整分析维度
- 支持追加新的结单数据

## 关键指标模板

### 储蓄账户分析
| 指标 | 计算方式 |
|------|----------|
| 总收入 | 所有存入金额累计 |
| 总支出 | 所有支出金额累计 |
| 净现金流 | 总收入 - 总支出 |
| 月均收入 | 总收入 ÷ 月数 |
| 月均支出 | 总支出 ÷ 月数 |
| 资金续航 | 当前余额 ÷ 月均支出 |

### 往来账户分析
| 指标 | 计算方式 |
|------|----------|
| 总转入 | 所有转入金额累计 |
| 总转出 | 所有转出金额累计 |
| USD 转出 | USD 账户转出总额 |
| 待查金额 | 无法追踪去向的资金 |

## 注意事项

1. **数据准确性**
   - 所有数据基于用户提供的结单
   - 支出分类为估算比例，实际需根据交易明细确认
   - 大额交易需用户核实

2. **隐私保护**
   - 不泄露账户详细信息给第三方
   - 敏感数据（如完整账号）需脱敏处理

3. **待查事项追踪**
   - 往来账户大额转出需标记待查
   - 提供核查步骤建议

4. **文件格式**
   - PDF 使用 HTML 格式避免字体问题
   - Excel 支持直接编辑和筛选
   - HTML 支持浏览器查看和打印
