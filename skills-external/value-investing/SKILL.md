# 价值投资分析技能（巴菲特风格）

## 描述
基于巴菲特价值投资理念，分析股票的投资价值，发掘适合长期持有（5 年以上）的优质个股。

**触发条件：**
- 用户询问某只股票是否值得投资
- 用户要求发掘价值投资标的
- 用户要求分析行业数据

**投资理念：**
- 长期持有（至少 5 年）
- 投资有价值的企业，而非炒作股票
- 关注企业基本面，而非短期股价波动
- 安全边际原则
- 能力圈原则

---

## 巴菲特价值投资框架

### 一、定性分析（50% 权重）

#### 1. 商业模式（15 分）
- [ ] 业务是否简单易懂？
- [ ] 是否有清晰的盈利模式？
- [ ] 是否在自己的能力圈内？

#### 2. 护城河（20 分）
- [ ] 是否有品牌优势？
- [ ] 是否有成本优势？
- [ ] 是否有网络效应？
- [ ] 是否有高转换成本？
- [ ] 是否有特许经营权/专利？

#### 3. 管理层（15 分）
- [ ] 管理层是否诚信？
- [ ] 管理层是否有能力？
- [ ] 管理层是否以股东利益为导向？
- [ ] 管理层持股情况如何？

#### 4. 行业前景（10 分）
- [ ] 行业是否处于成长期？
- [ ] 行业竞争格局如何？
- [ ] 行业是否有长期发展空间？

### 二、定量分析（50% 权重）

#### 1. 盈利能力（20 分）
- [ ] ROE（净资产收益率）> 15%（连续 5 年）
- [ ] 毛利率 > 30%
- [ ] 净利率 > 10%
- [ ] 盈利是否稳定增长？

#### 2. 财务健康（15 分）
- [ ] 资产负债率 < 50%
- [ ] 流动比率 > 1.5
- [ ] 自由现金流是否充沛？
- [ ] 是否有大额商誉风险？

#### 3. 成长性（10 分）
- [ ] 营收增长率 > 10%
- [ ] 净利润增长率 > 10%
- [ ] 增长是否可持续？

#### 4. 估值水平（15 分）
- [ ] PE（市盈率）< 25 或低于行业平均
- [ ] PB（市净率）< 5 或低于行业平均
- [ ] PEG < 1.5
- [ ] 是否有安全边际（>30%）？

### 三、分红政策（10% 权重）

#### 5. 股东回报（10 分）
- [ ] 是否有稳定分红历史？
- [ ] 股息率 > 2%？
- [ ] 分红比例是否合理？

---

## 评分标准

| 总分 | 评级 | 建议 |
|------|------|------|
| 90-100 | ⭐⭐⭐⭐⭐ 强烈推荐 | 符合巴菲特标准，可重仓 |
| 80-89 | ⭐⭐⭐⭐ 推荐 | 优质标的，可配置 |
| 70-79 | ⭐⭐⭐ 谨慎推荐 | 有亮点但有风险 |
| 60-69 | ⭐⭐ 观望 | 等待更好价格 |
| <60 | ⭐ 不推荐 | 不符合价值投资标准 |

## 执行步骤

### 步骤 1：收集数据
- 财务数据（年报/季报）
- 行业数据
- 竞争对手数据
- 新闻资讯

### 步骤 2：定性分析
- 商业模式分析
- 护城河评估
- 管理层评估
- 行业前景分析

### 步骤 3：定量分析
- 计算财务指标
- 与历史数据对比
- 与同行业对比
- 估值分析

### 步骤 4：综合评分
- 计算各项得分
- 给出综合评级
- 提供投资建议

### 步骤 5：生成报告
- 投资分析报告（PDF/HTML）
- 关键数据表（Excel）
- 风险提示

## 注意事项

1. **不是投资建议**：分析仅供参考，不构成投资建议
2. **风险自担**：投资有风险，入市需谨慎
3. **长期视角**：价值投资需要耐心，短期波动正常
4. **能力圈**：只投资自己理解的企业
5. **安全边际**：价格低于价值时买入
