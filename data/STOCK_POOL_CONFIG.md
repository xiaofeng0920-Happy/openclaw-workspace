# 📊 巴菲特股票池配置文档

> 筛选符合巴菲特标准的优质股票池

---

## 🎯 筛选标准

### 基础条件

| 条件 | 要求 | 说明 |
|------|------|------|
| **市场** | A 股 / 港股 / 美股 | 全球主要市场 |
| **非 ST** | ✅ | 排除风险警示股票 |
| **市值** | ≥ 50 亿 | 大盘股，流动性好 |

### 财务指标（连续 5 年）

| 指标 | 要求 | 说明 |
|------|------|------|
| **股息率** | ≥ 5% | 持续分红能力 |
| **ROE** | ≥ 10% | 股东权益回报率 |
| **ROIC** | ≥ 10% | 投入资本回报率 |

---

## 📈 筛选流程

```
1. 获取股票列表
   ↓
2. 过滤 ST 股票
   ↓
3. 过滤市值<50 亿
   ↓
4. 检查股息率 (5 年≥5%)
   ↓
5. 检查 ROE(5 年≥10%)
   ↓
6. 检查 ROIC(5 年≥10%)
   ↓
7. 生成股票池
```

## 🎯 买入信号监控

### 监控条件

| 条件 | 阈值 | 优先级 |
|------|------|--------|
| 安全边际 | ≥ 30% | 🔴 高 |
| 巴菲特评分 | ≥ 80 分 | 🟡 中 |
| 跌破目标价 | ≤ 设定价 | 🔴 高 |
| 严重低估 | < 内在价值 70% | 🔴 高 |
| 超卖信号 | 单日跌幅>5% | 🟢 低 |

### 监控频率

**交易日：**
- 09:00 盘前检查
- 09:30-16:00 每 30 分钟
- 16:30 盘后总结

### 通知方式

触发时自动发送飞书消息：
- 🎯 股票名称和代码
- 📈 当前价格和内在价值
- 📊 安全边际和评分
- 💡 投资建议
- 💵 建议仓位

## ⚠️ 注意事项

### 数据限制

1. **ROIC 数据**: 部分市场数据不完整
2. **5 年数据**: 新股可能数据不足
3. **实时性**: 财务数据有延迟

### 筛选调整

可根据需要调整筛选条件：

```python
# 编辑 stock_screener.py
class BuffettStockScreener:
    MIN_MARKET_CAP = 50       # 最小市值 (亿)
    MIN_DIVIDEND_YIELD = 5.0  # 最小股息率 (%)
    MIN_ROE = 10.0            # 最小 ROE (%)
    MIN_ROIC = 10.0           # 最小 ROIC (%)
    MIN_YEARS = 5             # 连续年数
```
