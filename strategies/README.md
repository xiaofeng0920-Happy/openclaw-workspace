# 📊 巴菲特价值投资系统

> "价格是你付出的，价值是你得到的。" —— 沃伦·巴菲特

---

## 📖 系统概述

本系统将沃伦·巴菲特的投资哲学量化为可执行的选股和交易策略，结合富途 OpenAPI 实时数据，为投资者提供基于价值投资理念的决策支持。

### 核心理念

1. **护城河理论** - 投资具有持续竞争优势的企业
2. **安全边际** - 只在价格大幅低于内在价值时买入
3. **长期持有** - 买入后至少持有 3-5 年
4. **集中投资** - 重仓最有信心的股票
5. **能力圈** - 投资自己理解的业务

## 📐 巴菲特选股标准

### 量化指标

| 指标 | 巴菲特标准 | 权重 |
|------|-----------|------|
| **ROE (净资产收益率)** | > 15% 持续 5 年 | 25% |
| **盈利增长率** | > 10% 年化 | 20% |
| **负债/权益** | < 0.5 | 15% |
| **毛利率** | > 30% | 10% |
| **护城河评分** | > 6/10 | 20% |
| **市盈率 PE** | < 15 | 10% |

### 评分体系

综合评分 = ROE 评分 (25) + 增长评分 (20) + 财务健康 (15) + 护城河 (20) + 估值 (20)

| 评分 | 等级 | 投资建议 |
|------|------|---------|
| 80-100 | ⭐⭐⭐⭐⭐ 优秀 | 强力买入 |
| 70-79 | ⭐⭐⭐⭐ 良好 | 买入 |
| 60-69 | ⭐⭐⭐ 中等 | 持有 |
| 50-59 | ⭐⭐ 较差 | 减持 |
| < 50 | ⭐ 差 | 卖出 |

## 💰 内在价值计算

### 方法一：DCF 模型（现金流折现）

```python
fair_value = Σ(未来现金流 / (1 + 折现率)^年数) + 终值 / (1 + 折现率)^n

参数:
- 折现率：10%（巴菲特常用）
- 永续增长率：2-3%（GDP 增速）
- 预测期：5 年
```

### 方法二：合理 PE 法

```python
fair_value = 每股收益 × 合理 PE

合理 PE = 15（巴菲特认为的合理估值）
```

### 安全边际计算

```python
安全边际 = (内在价值 - 当前价格) / 内在价值 × 100%

巴菲特建议：至少 30% 的安全边际
```

## 🎯 交易策略

### 买入规则

| 安全边际 | 评分 | 动作 | 仓位 |
|---------|------|------|------|
| > 30% | > 80 | 🎯 强力买入 | 20-40% |
| 20-30% | > 70 | ✅ 买入 | 10-20% |
| 10-20% | > 60 | ⏸️ 观望 | 维持 |
| < 10% | - | ❌ 不买入 | - |

### 卖出规则

| 溢价 | 动作 | 说明 |
|------|------|------|
| > 50% | 🔴 强力卖出 | 严重高估 |
| 30-50% | ⚠️ 卖出 | 高估 |
| 基本面恶化 | ⚠️ 减持/卖出 | ROE 大幅下降、负债激增 |

### 仓位管理

```
单只股票最高仓位：40%
单只股票最低仓位：5%
建议持股数量：5-10 只
```

## 🔧 自定义配置

### 调整选股阈值

编辑 `buffett_strategy.py`:

```python
class BuffettStrategy:
    MIN_ROE = 15.0  # 最低 ROE
    MAX_DEBT_TO_EQUITY = 0.5  # 最高负债比
    MAX_PE = 15.0  # 最高 PE
    MIN_MARGIN_OF_SAFETY = 30.0  # 最低安全边际
```

### 添加新股票数据

编辑 `buffett_analyzer.py` 的 `estimate_financials()` 函数:

```python
estimates = {
    'HK.00001': {  # 新股票代码
        'roe': 18.0,
        'roe_5y_avg': 16.5,
        # ... 其他财务指标
    },
}
```

## ⚠️ 风险提示

1. **数据准确性**: 财务数据可能存在延迟或误差
2. **模型局限**: DCF 模型对假设敏感，结果仅供参考
3. **市场风险**: 价值投资需要长期持有，短期可能波动
4. **个人判断**: 系统建议不替代个人独立研究
